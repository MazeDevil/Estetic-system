# Estetic - Clinica Estetica

Sistema web estruturado para o projeto interdisciplinar de Banco de Dados e Programacao Web II.

## Tecnologias

- Node.js
- Express
- Handlebars
- MySQL
- HTML, CSS e JavaScript

## Como executar

1. Instale as dependencias:

```bash
npm install
```

2. Crie o arquivo `.env` com base em `.env.example`.

3. Execute o SQL em `sql/BD.sql` no MySQL Workbench.

4. Inicie a aplicacao:

```bash
npm run dev
```

5. Acesse `http://localhost:3000`.
